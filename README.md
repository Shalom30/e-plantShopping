# Paradise Nursery 🌿

## Project Name: Paradise Nursery Shopping Application

A React + Redux shopping cart application for an online plant shop offering a wide variety of houseplants.

## Features

- **Landing Page** – Welcome screen with company branding and a "Get Started" button
- **Product Listing Page** – Browse 18+ unique houseplants grouped into 3 categories
- **Shopping Cart Page** – Manage items, adjust quantities, see totals
- **Redux State Management** – Cart state managed globally with Redux Toolkit
- **Responsive Navbar** – With live cart item count

## Plant Categories

1. Aromatic Plants
2. Medicinal Plants
3. Air-Purifying Plants

## Tech Stack

- React 18
- Redux Toolkit
- React Router DOM
- Vite

## Getting Started

```bash
npm install
npm run dev
```
